## Start

<a href="https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/">install python and pip</a>

```bash
  
  linux or unix
  cd execl_reader
  python3 -m venv env
  source env/bin/activate
  pip install -r requirements.txt
  python run.py

  windows
  cd execl_reader
  py -m venv env
  .\env\Scripts\activate
  pip install -r requirements.txt
  py run.py

  

```
